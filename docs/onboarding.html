Generate your personal CONTEXT.md

Before building, create your study context file. Open Claude.ai or the Claude desktop app and paste this prompt. Claude will interview you and produce a ready-to-save docs/CONTEXT.md.

I'm preparing for the CCA-F (Claude Certified Associate – Foundations) certification exam and starting the AgentCX project. Please interview me to build my personal CONTEXT.md file. Ask me about: 1. My background and experience level with Claude SDK and LLMs 2. Which CCA-F domains I've already studied (D1–D5) 3. Mock exam results, if any (score, domain breakdown, weak areas) 4. Key concepts I keep getting wrong under pressure After the interview, generate a complete CONTEXT.md file in this exact format: - Header with my name and background - Exam structure table (use the standard CCA-F domain weights) - Study progress per domain (✅ Complete / 🔄 In progress / ⬜ Not started) - Exam results table if applicable - Key concepts to reinforce section - A "How to continue" section that says: start a Claude Code session and run /agentcx-builder I will save this file to docs/CONTEXT.md in the agentcx repo.
Then: Save the output as docs/CONTEXT.md in your cloned repo. The builder skill reads this file at the start of every session to know where you left off.
1

Clone the repo — starter branch

The starter branch has no implementation code — just the docs and the builder skill. You'll build everything yourself.

git clone https://github.com/JavM1212/agentcx.git
cd agentcx
git checkout starter
2

Create your virtual environment

Use Python 3.10+ to avoid type hint compatibility issues.

python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
Tip: Add this alias to your ~/.zshrc so you can run any module with run <module>:

alias run='/path/to/agentcx/.venv/bin/python -m'
3

Add your API key

Create a .env file in the project root. Never commit this file — it's already in .gitignore.

# .env
ANTHROPIC_API_KEY=sk-ant-...
4

Open Claude Code

Launch Claude Code from the project root.

claude
5

Start building

Invoke the builder skill. Claude will read the reference docs, detect where you are, and guide you through each step.

/agentcx-builder
What to expect: Claude explains concepts with analogies, asks you questions before writing code, introduces intentional bugs for you to find, and connects every decision to a CCA-F exam task statement. You control the pace.
